# Initial Rebuttal Style

For the first rebuttal round, prefer:

- a short global opener
- direct answer before long framing
- one strong point per paragraph
- shared issues surfaced early
- reviewer-specific emphasis after the global strategy is clear

Avoid:

- repetitive gratitude loops
- broad promises
- hiding the main answer after too much setup

Under tight character limits, prioritize:

1. the strongest shared issue
2. swing reviewers
3. narrow and evidence-bound claims
